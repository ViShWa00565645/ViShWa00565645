##  My First Project

### BehindLens AI

BehindLens AI is a web application that helps users create professional-looking photos with AI-powered editing tools.

### Features

*  Text Behind Image Effect
*  DSLR Blur Effect
*  Inspiration Gallery for creative ideas
*  Fast and easy-to-use interface
*  Mobile-friendly design

🔗 Live Demo: https://poster-ai-pro-mpe9.onrender.com/

### Why I Built It

I wanted to create a simple photo editing experience where users can generate modern social-media-style images without using complicated editing software.
